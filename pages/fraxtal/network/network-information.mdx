## Other RPC Providers

### Tenderly

[Tenderly](https://tenderly.co/web3-gateway) provides both HTTP and WebSocket access to Fraxtal and offers additional custom RPC methods to help debug/simulate requests.

The public, rate-limited RPC URL is `(https|wss)://fraxtal.gateway.tenderly.co` but you can also generate an API key for additional throughput using your Tenderly account.

### Grove

[Grove](https://www.grove.city) provides RPC access backed by thousands of independent, globally distributed nodes from the [POKT](https://www.pokt.network) Network.

### Nirvana

[Nirvana](https://nirvanalabs.io) provides private, dedicated Fraxtal nodes on their optimized bare metal cloud.

### Opsec

[Opsec](https://opsec.computer) Hosting Fraxtal nodes on the OpSec network empowers users to actively contribute to the decentralization and strengthening of the Fraxtal mainnet.

### dRPC

[dRPC](https://drpc.org/chainlist/fraxtal?utm_source=docs&utm_medium=fraxtal) Access a network of distriputed Fraxtal RPC nodes through dRPC platform.
